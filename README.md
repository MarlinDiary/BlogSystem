## 安装指南

### 后端配置
1. 进入后端目录
```bash
cd backend
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量（创建 .env 文件）
```
PORT=3000
SERVER_URL=http://localhost:3000
```

### 前端配置
1. 进入前端目录
```bash
cd frontend
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量（创建 .env 文件）
```
VITE_OPENAI_API_KEY=your_openai_api_key_here
PUBLIC_API_URL=http://localhost:3000
```
